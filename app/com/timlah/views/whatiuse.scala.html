This Website

Scala Scala's my main language at work as of the time of writing - So, I wanted a website that I could build in the same language to help continually grow in this ecosystem.
Play Framework My go-to framework for building a website in Scala. Session management, templating, configuration and more out of the box makes this a powerhouse.
Slick Great library for connecting to databases.
Laika Markdown made easy with Scala - Powers the interpreting of blog posts.
Heroku I wanted a very simple, quick build tool that would support Scala. Heroku, whilst not the cheapest, also provides databases and schedulers with minimal configuration.
node.js Accessibility tests using axe-core, pulling the dependency in via npm and writing JavaScript tests for this is simple and flexible.
Google Analytics Simple, well known analytics platform. Let's me know that people have seen my stuff and I'm not just screaming into the void.
Manjari (Font) I mean, you're reading the website in the Manjari font. What do you think?
ko-fi No fuss, simple way for people to donate towards the coffee fund.
Disqus Allows me to have a simple, working comment system, without having to manage the hosting of the content myself.

Software choices

Technology Decision for choice Images, links, further info, etc
VS Code Great support for Scala (Metals) and easily jump between different languages, set up the way I like. Great extensions and community. Plus it looks nice with the `Dobri Next -A00- Black` theme.
Obsidian Helps me keep notes - and automated to allow me to create a daily todo list, displayed on my desktop.
Windows Desktop My desktop environment is in Windows, where I leverage WSL (Windows Subsystem for Linux) to do some tasks.

Linux Laptop (Ubuntu) I like Linux, it gives me all I need for development. Quick, reliable and Ubuntu is a good all-rounder for my needs. Your mileage may vary.
Unity For game development, I still really like Unity, as it gives you a lot of power upfront.
GIMP (GNU Image Manipulation Program) For editing of any images, sprites etc, this is a great free alternative to Photoshop.
